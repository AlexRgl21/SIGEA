# SIGEA - Sistema Integral de Gestión de Espacios Académicos

Plataforma web desarrollada para optimizar la administración y asignación de aulas, laboratorios y espacios académicos. Su objetivo principal es automatizar el control de horarios y evitar empalmes mediante un Motor de Conflictos en tiempo real.

## Características Principales
* **Gestión de Espacios:** Catálogo completo de edificios y salones.
* **Control de Horarios:** Asignación de grupos a espacios específicos por bloques de tiempo.
* **Motor de Prevención de Empalmes:** Algoritmo que bloquea asignaciones si el espacio ya se encuentra ocupado en ese día y rango horario.
* **Interfaz Intuitiva:** Dashboard visual y responsivo para los coordinadores académicos.
